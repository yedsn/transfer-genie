## ADDED Requirements

### Requirement: 已标记消息筛选器
应用程序 SHALL 在消息列表的标题中显示一个筛选控件。该控件 SHALL 显示已标记消息的总数。单击该控件 SHALL 切换消息流中未标记消息的可见性。

#### 场景：查看已标记消息计数
- **假如** 有 3 条已标记消息
- **那么** 筛选控件应显示数字“3”。
- **当** 一条消息被标记时
- **那么** 筛选控件上的计数应增加到“4”。
- **当** 一条消息被取消标记时
- **那么** 筛选控件上的计数应减少到“3”。

#### 场景：筛选已标记消息
- **假如** 消息列表同时显示已标记和未标记的消息
- **当** 用户单击已标记消息筛选控件时
- **那么** 消息列表应更新以仅显示已标记的消息
- **并且** 筛选控件应指示活动的筛选状态。

#### 场景：取消筛选已标记消息
- **假如** 消息列表当前已筛选为仅显示已标记的消息
- **当** 用户再次单击已标记消息筛选控件时
- **那么** 消息列表应更新以显示所有消息
- **并且** 筛选控件应返回其默认状态。
