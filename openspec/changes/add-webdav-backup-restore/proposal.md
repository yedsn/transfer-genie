# 变更：新增 WebDAV 备份与恢复功能

本次变更将引入 WebDAV 数据的备份与恢复功能。备份过程将创建包含 `files/` 目录与 `history.json` 的 zip 压缩包。恢复过程会在覆盖现有 WebDAV 数据前提示用户确认。

## 动机

用户需要一种方式来备份数据以确保安全，并在不同 WebDAV 服务器之间迁移数据。这提供了一个安全保障，并增强了用户数据的可移植性。

## 范围

-   在客户端设置界面新增“备份 WebDAV”与“恢复 WebDAV”按钮。
-   实现后端逻辑，将 WebDAV 的 `files/` 目录与 `history.json` 文件流式压缩成 zip 包用于备份。
-   实现后端逻辑，解压 zip 包并将其内容上传至 WebDAV 用于恢复。
-   在恢复操作覆盖数据前，增加一个确认对话框。

## 范围之外

-   自动备份。
-   部分备份或恢复。
-   应用程序设置的备份/恢复（这由现有的配置导入/导出功能处理）。
