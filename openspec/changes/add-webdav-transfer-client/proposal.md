# 变更：新增 WebDAV 传输客户端

## 为什么
构建一个跨平台传输助手，使用 WebDAV 作为共享后端，让多台设备可以发送文件或文本，并在类似聊天的列表中看到更新。

## 变更内容
- 新增基于 WebDAV 文件的消息列表，并使用本地 SQLite 建立索引。
- 新增 WebDAV 连接/认证、发送者名称、刷新间隔与手动刷新设置。
- 新增聊天式界面，可发送文本与文件，并展示发送者、时间与大小。

## 影响范围
- 受影响规格：message-feed、client-settings
- 受影响代码：新的 Tauri 应用（前端 + 后端）、WebDAV 客户端集成、本地 SQLite 存储
